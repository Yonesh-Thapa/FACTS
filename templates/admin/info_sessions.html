Step 1: Create Info Session
  • Click "Manage Info Sessions"
  • Add a new info session with date, time and Zoom details

  • Select the info session from the dropdown
  • Add optional custom message
  • Click "Send Zoom Links to All"
  • Links only go to people who haven't received them

  • Reminders only go to people who received Zoom links
